# composer
This is docker image to run the [composer](https://getcomposer.org).

Image size: 43 MB

From image: alpine:3.1

Composer version: 1.0.0-alpha10

## Usage

```
$ docker run --rm -v /path/to/project/:/data imega/composer install --no-dev
```
